HOSPITAL WEB APP - LOCAL INSTALLATION GUIDE
============================================

You do NOT need Flutter, Node.js, or Python installed.


HOW TO RUN THE APP
------------------
1. Unzip this entire folder anywhere on your computer.
2. Double-click:  Launch Hospital App.bat
3. Your browser will open automatically.
4. Keep the black "Hospital App Server" window open while you use the app.
5. Close that server window when you are finished.


DO NOT DO THIS
--------------
- Do not double-click index.html. It will show a blank page.
- This app must run through the launcher batch file.


ADD A DESKTOP ICON (RECOMMENDED)
--------------------------------
Option A - Shortcut to launcher:
1. Right-click "Launch Hospital App.bat"
2. Click "Show more options" (if shown)
3. Click "Send to" -> "Desktop (create shortcut)"
4. Rename the shortcut to "Hospital App"
5. Double-click that desktop icon each time you want to open the app.

Option B - Install as Chrome app (optional):
1. Run the app using "Launch Hospital App.bat"
2. In Google Chrome, click the install icon in the address bar
   (or open Chrome menu -> "Install USMAN CHAUDHARI...")
3. This adds an app icon in Windows Start and on the desktop.

Note: For Option B, the server window from the launcher must still be running
when you first open the installed app. After that, Chrome may cache files for
offline use, but using the desktop shortcut from Option A is the most reliable.


TROUBLESHOOTING
---------------
- Blank page: Make sure you used "Launch Hospital App.bat", not index.html.
- Port in use: Close any other copy of the app, then try again.
- Windows security warning: Click "More info" -> "Run anyway" if Windows blocks
  the launcher the first time.
- Use Google Chrome or Microsoft Edge for best results.
